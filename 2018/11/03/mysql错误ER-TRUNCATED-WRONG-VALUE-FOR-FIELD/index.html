ER_TRUNCATED_WRONG_VALUE_FOR_FIELD: Incorrect string value

Mac系统下mysql报错：ER_TRUNCATED_WRONG_VALUE_FOR_FIELD: Incorrect string value

这种原因是mysql字符集编码导致。

  1. 查看 mysql 字符集编码
    进入mysql，终端执行
    $ mysql -u root -p
    登录成功后在mysql中执行
    $ show variables like '%character%';
