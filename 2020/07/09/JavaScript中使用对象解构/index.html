8. Rest 和 Spread 运算符

我们经常会混淆 Rest 和 Spread 运算符，因为它们都使用相同的 ... 语法。Rest 和 Spread 运算符可以在 Object Destructuring 语句中一起使用:

  • Rest 操作符: 用于语句的 左边，用于从对象获取剩余属性
  • Spread 操作符: 用于语句的 右侧，用于将属性复制到对象
